R 程式設計班

本課程希望可以結合 SQL 與 R 語言，讓商業分析師（非資訊部門員工）能獨立完成從資料抓取到分析的整個流程，進而提升對工作的掌握度。

課程資訊

  • 寒暑假密集班
  • 夜間班

課程大綱

  • 建立你的 R 語言開發環境
  • 單一變數與函數
  • 簡單的繪圖（Base plotting system）
  • 基本資料結構
  • 簡單的繪圖（ggplot2）
  • 用 RMarkdown 撰寫報告
  • 讀取不同的資料格式
  • 活用資料框
  • 迴圈與流程控制
  • 使用 apply 函數家族
  • 自訂函數
  • 使用 %>% 簡化程式
  • 地理資訊的繪圖（Leaflet）
  • 使用 dplyr 套件整理資料
  • 迴歸模型（Regression）
  • 分群模型（Clustering）
  • 分類模型（Classification）
  • 參加 Kaggle 競賽
