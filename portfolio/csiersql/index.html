SQL 及 R 語言的資料分析實戰

本課程希望可以結合 SQL 與 R 語言，讓商業分析師（非資訊部門員工）能獨立完成從資料抓取到分析的整個流程，進而提升對工作的掌握度。

適合修課對象

  • 想學習 SQL 與 R 語言的產品、行銷與金融分析從業人士
  • 未接觸過程式的初學者

課程大綱

  • SQL 基礎
  • SQL 進階
  • 建立 R 語言的開發環境
  • 建立 R 語言與資料庫的連結
  • 讀取不同的資料格式
  • 簡單的繪圖（Base plotting system）
  • dplyr 套件
  • 簡單的繪圖（ggplot2）
  • 用 RMarkdown 撰寫報告
