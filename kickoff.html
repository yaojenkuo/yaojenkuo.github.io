Churn Prediction - Kickoff

Tony Kuo

June 23, 2016

Request

  • We want every customer to have a score that indicates how likely he or she is to churn during the next 6-12 months
  • Inputs to this score should include both purchase history and engagement history, and maybe other factors
  • The score should be constantly updated so that marketing channel owners can use the information as an input to campaign and feature design
  • Random forest analysis is recommended

Models to be Adopted

  • Binary classification
    • Logistic regression
    • Random forest(recommended)
  • Survival analysis

Predictors to be Discussed

  • Purchase history data, including order pattern, order recency, categories purchased, cancellations, coupon usage, Coupang Cash usage, and use of Coupang exclusive services
  • Engagement data, including visit pattern, visit recency, visit channels (e.g.paid, organic, app push, email), visit platforms, application version, and opt-in to CRM programs
  • Acquisition data, including purchase history information tied to their first purchase (above), date of first purchase, date of install, acquisition channel (e.g.Organic, Paid Display Ads, Paid Search, etc.), and acquisition publisher (e.g.Cauly Display Ads, Naver Search)
  • Demographic data, including age, gender, zip code, and zip-code based roll ups (e.g.rural vs.urban)

Data Sources to be Discussed

  • PDA tables
  • Hive tables
  • SCI(Customer Science Interface)
  • etc.

# Use the survival library
library(survival)
# Create a "survival object" for each observation, using time and churn data.
demo$survival <- Surv(demo$time, demo$churned == 1)
# Fit a basic survival curve using the data
fit <- survfit(survival ~ 1, data = demo)
# Plot the survival curve and add a title!
plot(fit, lty = 1, mark.time = FALSE, ylim=c(.75,1), xlab = 'Days since Last Purchase', ylab = 'Percent Surviving')
title(main = 'Demo Survival Curve')
abline(v = 182, lty = 2, col = "red", lwd = 2)
abline(h = 0.885, lty = 2, col = "green", lwd = 2)

We can expect 11% of customers will be churned given our definition of 182 days without a single purchase.

# Fit survival curves based on the female binary
fit <- survfit(survival ~ female, data = demo)
# Plot the results, add a legend, add a title
plot(fit, lty = 1:2, mark.time = FALSE, ylim=c(.75,1), xlab = 'Days since Last Purchase', ylab = 'Percent Surviving')
legend(20, .8, c('Male', 'Female'), lty=1:2, bty = 'n', ncol = 2)
title(main = "Demo Survival Curves by Gender")
abline(v = 182, lty = 2, col = "red", lwd = 2)
